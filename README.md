## Client Usage (C#)
```
1. 서버에서 발급받은 ID, PWD로 스크린 세이버 화면을 잠금 해제한다.
2. 관리자가 부여한 사용자 권한에 따라 기본 프로그램 사용이 제한된다. (CMD, Regedit, 작업관리자, 캡처 도구 등)
3. ITF 프로그램을 실행하여 사용. (파일 암복호화, 화면 잠금 타이머 등 기능이 포함되어 있다.)
```
